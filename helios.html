BIY-Build it yourself

Jan. 2016 - Mar. 2016 (2 months)

Project Description

This project implements a prototype Continuous Integeration System based on Google App Engine, Google Container Engine. It leverages a lot of cool up-to-date technologies including Cloud stoarge, Datastore, TaskQueue and memcache etc.

Architecture Overview

For the backend, I use Jersey as a RESTful service provider to provide necessary information to be consumed by the frontend. The front end is made by AngularJS framework for the interactive user experience. Here is the technology stack.

Backend was totally RESTful and no user state is maintained. Here are the REST API endpoints.

Core Functions

Here are the core functions of system.

  • Login via Github OAuth2. Sync Git repoistories automatically.
  • Add or remove push event web hook within one-click.
  • Project automatically built after a push event. Continuously fecthing console log to the output panel.
